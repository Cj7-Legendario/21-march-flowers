# 21-march-flowers
# 🌻 Proyecto 21 de Marzo: Flores Eternas de Van Gogh

Este no es un simple script de felicitación. Es una **experiencia interactiva de alto rendimiento** diseñada para el 21 de marzo, combinando el arte clásico de Vincent van Gogh con animaciones fluidas en Canvas y CSS.

## 🚀 Características "Locasas"
- **Terminal de Acceso:** Simulación de hackeo con verificación de identidad.
- **High Performance:** Renderizado optimizado a 60fps sin librerías pesadas.
- **Van Gogh Reveal:** El ramo de girasoles se dibuja dinámicamente usando máscaras de Canvas.
- **Modo Captura (📸):** Interfaz limpiable para tomar el screenshot perfecto.
- **Status Bar 2026:** Indicador de sistema y reloj en tiempo real.

## 🛠️ Tech Stack
- **HTML5 / CSS3** (Animaciones nativas)
- **JavaScript Moderno** (Lógica de estados y manipulación de DOM)
- **Canvas API** (Para el efecto de revelado del ramo)
